Box Model

Web browsers represents each element on a web page as a rectangular box according to the standard CSS basic box model.
A CSS box model is used when talking about layout and design
The CSS box model is a box that wraps around every HTML element.It is a set of rules that defines how every element in a web page is rendered.
Its properties consists of:

  1. content-This include text, images,videos,audios etc
  2. padding-The space surroundeing the content. It seperates content from the border or margin. It is transparent.
  3. border-The line surrounding the content and the padding separating them from the margin
  4. margin-It clears an area outside the border.
  5. width- the width
  6. height-the height

Calculating the total Dimension of a box model:
width= width + padding-left + padding-right + border-left + border-right
